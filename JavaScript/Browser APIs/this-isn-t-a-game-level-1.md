## Content

Do you know how `this` works in JavaScript? What is `this` in each context? Assume the code is run in a browser (not in a Worker).

---
## Game Content

```
console.log(this); 
```
* `window`
* `console`
* `undefined`

```
function f(){
  console.log(this);
}
f();
```
* `window`

```
function f(){
  "use strict";
  console.log(this);
}
f();
```
* `undefined`
* `window`
* `f`

```
var user = {
    firstName:"Anne",
    lastName:"Boleyn",
    showName:function(){
     console.log(this.firstName + 
             " " + this.lastName);
     }
    }
user.showName();
```
* `user`
* `window`
* `undefined`
---
```
var obj = {add: function(){
            return this.x + this.y;
           }};
var sum = Object.create(obj);
sum.x = 3;
sum.y = 7;
console.log(sum.add());
```
* `sum`
* `obj`
* `window`
